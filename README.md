# React Typescript Template

This is a template for a React project with Typescript, Tailwind, HeadlessUI, Heroicons and React Router Dom all preinstalled.

## Getting Started

To get started, you can clone this repository to get a copy of the template.

## Available Scripts

In the project directory, you can run:

### `npm start`

Runs the app in the development mode.\

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
